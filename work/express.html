Client Overview

Express is an apparel brand inspiring confidence and self-expression. They operate 400+ stores across the U.S. and Puerto Rico, plus e-commerce and mobile channels delivering modern style for work, everyday, and special occasions.

With NuLogic's partnership since 2012, Express needed a modern, agile cloud foundation to fuel digital growth and compete in fast-moving retail.

The Challenge

Legacy Infrastructure Holding Back Growth

Aging AWS environment slowed releases and limited innovation velocity needed to compete in fast-moving retail.

Manual DevOps Bottlenecks

Legacy Jenkins pipelines and manual infrastructure provisioning created errors, delays, and reduced visibility into system performance.

Search Experience Gap

Generic product search lacked personalization and contextual understanding needed to drive engagement and conversions.

The Solution

Pillar 1: Cloud Migration (AWS → Google Cloud)

Executed zero-downtime migration using Terraform for infrastructure automation. Provisioned GCP resources (IAM, networking, GKE clusters, storage) with code-driven precision, eliminating manual provisioning errors.

Pillar 2: DevOps Automation & GitOps

Transitioned to ArgoCD and GitHub Actions for continuous deployment. Replaced legacy Jenkins with Argo Workflows. Migrated Kubernetes management from Rancher to ArgoCD for streamlined, declarative container orchestration across environments.

Pillar 3: Intelligent Search & Recommendations

Deployed Vertex AI Search for Commerce, a fully managed ML-driven platform. Enabled semantic search, multi-modal results (text + image), and conversational refinement. Powered personalized recommendations using AI/ML algorithms on user behavior and intent—no in-house ML pipelines required.

The Results

  • Zero-downtime Cloud Migration: Successfully moved from AWS to Google Cloud with no disruption to operations.
  • 40% Faster Release Cycles: Infrastructure-as-code and GitOps eliminated manual provisioning bottlenecks.
  • 99.9% Uptime: ArgoCD-managed Kubernetes and real-time Slack alerts ensure always-on reliability.
  • AI-Driven Conversions: Vertex AI Search delivered semantic, intent-based discovery boosting product engagement and conversion rates.
