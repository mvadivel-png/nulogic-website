Express Modernizes Cloud with Google, Drives Smarter Search with Vertex AI

Region

U.S.

Website

express.com

Client Since

2012

Client Overview

Express is an apparel brand dedicated to creating confidence and inspiring self-expression. Operating 400+ retail and outlet stores across the United States and Puerto Rico, plus e-commerce and mobile app channels, Express delivers modern, confident, effortless style across work, everyday, and special occasion wear.

The Challenge

Retail moves quickly, and legacy technology cannot power future growth. Express faced mounting pressures to modernize its eCommerce infrastructure while enhancing the online shopping experience.

Key initiatives included:

  • Google Cloud Platform migration to unlock agility, resilience, and cost efficiency
  • Automated DevOps pipelines to accelerate releases, reduce errors, and improve system visibility
  • AI-powered search for contextual, personalized product discovery driving engagement and conversions

The Solution

Cloud Modernization & Automation (AWS → GCP)

  • Leveraged Terraform automation to provision and manage critical Google Cloud resources (IAM, networking, GKE clusters, storage), eliminating manual provisioning and error risk
  • Transitioned CI/CD pipelines to GitOps workflows using ArgoCD and GitHub Actions for faster, repeatable, secure deployments
  • Replaced legacy Jenkins pipelines with Argo Workflows for scalability and streamlined operations
  • Migrated Kubernetes management from Rancher to ArgoCD for simplified container orchestration across multi-cloud environments

Vertex AI Search Integration

  • Implemented Vertex AI Search for Commerce, a fully managed machine learning-driven product search and recommendations engine
  • Enabled semantic search, multi-modal results (text + image), advanced filtering, and conversational refinement to improve user experience and product discoverability
  • Enhanced personalized product recommendations using AI/ML algorithms leveraging user behavior and intent to drive engagement and conversion rates
  • Delivered AI-driven relevance, contextual search, and recommendation orchestration without managing in-house ML pipelines

The Results

The Wins, Delivered:

  • Seamless Cloud Shift: Zero-downtime migration from AWS to Google Cloud
  • DevOps on Autopilot: Terraform + GitOps pipelines slashed manual work, boosted speed, and cut release cycles
  • Always-On Reliability: Kubernetes management streamlined with ArgoCD; real-time monitoring + Slack alerts maintain operational resilience
  • AI That Sells: Vertex AI Search unlocked semantic, intent-driven personalization — powering smarter discovery and higher conversions

This transformation delivered a scalable, cost-efficient, and agile cloud foundation to support Express's digital growth.
