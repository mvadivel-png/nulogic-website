Case Study

Express Modernizes Cloud with Google, Drives Smarter Search with Vertex AI

Express trusted NuLogic to strengthen its technology backbone and build a future-ready platform on Google Cloud investing in AI-driven tools and automation to accelerate innovation and personalize shopping.

Client

Express

Region

United States

Client Since

2012

Industry

Retail & Apparel

The Challenge

Express faced modernization pressures requiring infrastructure updates and shopping experience enhancements. Key initiatives included:

  • Google Cloud Platform migration: Unlock agility, resilience, and cost efficiency
  • Automated DevOps pipelines: Accelerate releases and reduce errors
  • AI-powered search: Deliver contextual, personalized product discovery

The Solution

Cloud Modernization (AWS → GCP)

  • Terraform automation for Google Cloud resource management
  • GitOps workflows using ArgoCD and GitHub Actions
  • Replaced Jenkins pipelines with Argo Workflows
  • Kubernetes management transition from Rancher to ArgoCD

Intelligent Search with Vertex AI

  • Implemented Vertex AI Search for Commerce
  • Enabled semantic search, multi-modal results, and conversational refinement
  • Enhanced personalized recommendations leveraging user behavior
  • Delivered AI-driven relevance without managing in-house ML pipelines

Results

Downtime during AWS-to-GCP migration

Faster

Release cycles with Terraform + GitOps

AI-Powered

Semantic search & personalization
