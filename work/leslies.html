Client Overview

Founded in 1963, Leslie's operates 1,000+ retail stores and an eCommerce platform serving residential and professional pool/spa customers nationwide.

"Misalignment between development, QA, and design teams, slowing eCommerce delivery cycles and increasing project risks."

Key Challenges Identified:

  • Scalability issues during peak traffic on SFCC platform
  • Slow page load times from bloated JavaScript and CSS bundles
  • High API latency reducing responsiveness
  • Mobile UX challenges requiring optimization
  • Fragile mobile workflows causing instability
  • Fragmented loyalty program experience lowering retention

The Solution

NULogic assembled performance specialists, SFCC experts, and product leaders using Agile execution and continuous delivery.

Backend Refactoring

  • Refactored SFCC controllers following best practices
  • Reduced API latency and improved responsiveness
  • Rebuilt modular Boomi integrations for stability

Frontend & Mobile Optimization

  • Improved Lighthouse performance scores from sub-50s to mid-80s
  • Eliminated render-blocking resources and compressed bundles
  • Streamlined SFCC-Boomi-Mobile data flows
  • Optimized eCDN with advanced caching

Figma-Led Product Engineering

  • Designed UX flows converted to JIRA sprints
  • Applied engineering processes from Adidas and L'Oréal
  • Full-stack development across QA, UAT, and production

Field Services Portal

  • Rebuilt Service Appointment Portal with dynamic Q&A logic
  • Integrated SFCC objects with Accruent via Boomi APIs

Loyalty Program 2.0

  • Transformed flat-point system into tiered rewards structure
  • Integrated third-party platforms via real-time APIs
  • Enabled live points tracking and redemption visibility

Checkout Overhaul

  • Redesigned entire checkout experience (frontend + backend)
  • Embedded Google Analytics 4 for cookieless tracking
