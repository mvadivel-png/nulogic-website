Client Overview

America's largest pool and spa retailer. 1,000+ stores. 60+ years of expertise. One digital platform serving millions of residential and commercial customers.

The Challenge: A legacy platform buckling under peak traffic, slow load times killing conversions, and a fragmented loyalty experience bleeding customers.

What We Solved:

  • Performance Crisis: Lighthouse scores below 50, bloated code bundles, API latency crushing mobile experience
  • Platform Instability: Salesforce Commerce Cloud scalability failures during peak season, fragile integrations causing downtime
  • Broken Customer Journey: Disjointed loyalty program, checkout friction, service booking chaos

How We Did It

1. Speed & Scalability Engineering

Rebuilt the foundation for peak performance.

  • Refactored Salesforce Commerce Cloud controllers, slashed API latency, compressed bundles
  • Lighthouse mobile scores surged from sub-50 to 85+ through surgical code optimization
  • Rebuilt Boomi integrations for zero-downtime stability during traffic spikes
  • Deployed advanced eCDN caching eliminating render-blocking bottlenecks

2. Customer Experience Transformation

Redesigned every touchpoint for conversion and retention.

  • Loyalty 2.0: Flat points → tiered rewards with real-time tracking and redemption
  • Service Portal: Dynamic appointment booking with Accruent integration via Boomi APIs
  • Checkout Redesign: Complete frontend/backend overhaul with GA4 cookieless tracking
  • Mobile-First UX: Streamlined SFCC-Boomi-Mobile data flows for seamless cross-platform experiences

3. Enterprise-Grade Operations

24/7 monitoring and rapid-response infrastructure.

  • Dynatrace RUM: Real-time behavioral mapping of every customer journey
  • Opsgenie + JIRA: <15-minute resolution SLAs with instant engineering escalation
  • Multi-Platform Monitoring: SFCC, Boomi, Manhattan OMS unified observability
