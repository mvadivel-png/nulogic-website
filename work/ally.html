Client Overview

The nation's largest all-digital bank. Leader in auto financing and digital financial services. Ally Financial powers deposits, securities, investment advisory, and insurance through cutting-edge cloud infrastructure. Goal: Replace legacy ETL with native Snowflake automation for real-time data delivery.

Challenge

  • • Cost & Infrastructure: High licensing fees with Matillion ETL on EC2, plus infrastructure overhead and limited flexibility.
  • • Development Velocity: Each new extract required additional custom coding, delaying delivery and straining limited developer capacity.
  • • Operational Risk: Complex troubleshooting without centralized monitoring, increasing risk and reducing visibility into data pipelines.

Solution

Build a fully Snowflake-native extraction framework replacing Matillion with automated, config-driven data pipelines. No external ETL. No licensing fees.

Pillar 1: Automation Engine

Python procedures and Snowflake Task scheduling handle all extractions. Data writes directly to S3. No external tools required.

Pillar 2: Config-Driven Flexibility

New extracts deployed instantly via configuration updates—zero code changes needed. Scales to hundreds of extracts without engineering overhead.

Pillar 3: Centralized Governance

Audit logs, monitoring tables, and dependency handling provide real-time visibility. Proactive alerts and rapid recovery reduce operational risk.

Results

100% Cost Elimination

Matillion fully retired—no more licensing or infrastructure fees.

Days to Minutes

New extract onboarding accelerated with zero code changes needed.

Developer Freedom

Freed resources redirected to AI-driven personalization and innovation.

  • ✓ Real-time observability: Centralized monitoring and audit logs provide complete pipeline visibility.
  • ✓ Future-ready scale: Framework handles new vendors, formats, and transformations without architectural changes.
  • ✓ Lower operational risk: Proactive alerting and dependency handling prevent data pipeline failures.
