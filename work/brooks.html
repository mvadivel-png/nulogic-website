Case Study

Brooks Brothers Enhances Security and Performance with Cloudflare + Salesforce Commerce Cloud

NULogic implemented a dual CDN strategy with Cloudflare layered over Salesforce's native CDN, strengthening security and performance while eliminating Akamai complexity.

Client

Brooks Brothers

Region

U.S.

Website

brooksbrothers.com

Client Since

2023

Overview

Brooks Brothers is a legendary lifestyle brand founded in 1818, operating 500+ stores across 45 countries with 30+ product categories including tailored clothing, sportswear, and accessories.

The Challenge

Brooks Brothers faced malicious bot traffic inflating metrics and infrastructure costs on their SFCC platform. Their existing Salesforce CDN + Akamai combination lacked advanced bot management, rate-limiting, and edge security. The retailer needed stronger defense without sacrificing stability during peak retail events.

The Solution

Architecture Flow:

Client → Cloudflare (security, edge logic) → Salesforce CDN → SFCC origin

Bot Management & Security

  • Advanced bot detection and WAF deployment
  • Rate-limiting and API shielding capabilities

Performance Optimization

  • Anycast routing and image optimization
  • Edge caching for global delivery
  • Salesforce CDN continues serving static assets

Resilience Features

  • Cloudflare Waiting Room for peak events
  • Failover logic for checkout stability

Operational Improvements

  • Cloudflare Workers for header rewrites and redirects
  • Cache-coherency strategy between Cloudflare and Salesforce
  • Akamai retirement reducing vendor complexity

Measurable Results

Security

Significantly reduced malicious bot traffic, protecting checkout flows and lowering infrastructure load

Performance

Faster global delivery with improved TTFB and cache hit ratios

Peak Resilience

Zero disruption during flash sales and seasonal events

Operational Flexibility

Business teams deploy edge logic without SFCC code pushes

Cost Reduction

Simplified vendor landscape with reduced complexity

Support

Proactive monitoring and global off-hours support through NULogic retainer
